A criação da Internet

(Voltar)

Criada pelo britânico Tim Berners-Lee, HTML significa HiperText Markup Language, traduzindo para o português: Linguagem de Marcação de Hipertexto. O HTML é a estruturá básica das páginas da web, ele permite inserir o conteúdo e estabelecer a estrutura de um website. Portanto, ele serve para organizar as informações das diversas páginas da web. Sem isso, o navegador não iria conseguir expor os textos como elementos ou carregar imagens e diversos conteúdos.

Os hipertextos são conjuntos de elementos que estão conectados. Podendo, mas não limitados a ser documentos, vídeos, palavras ou imagens. Quando se conectam, formam uma estrutura de informações que permite a transmissão de dados, salvando informações e agrupando conhecimentos.

Ao visitar uma página qualquer da internet, é possível reparar que existem distribuições diferentes de tamanhos para títulos, parágrafos, imagens, vídeos ou qualquer outro elemento. Essa organização é estruturada através do HTML. Quando a internet começou, era normal encontrar sites contendo apenas textos e imagens, com estrutura básica e sem muito estilo. Porém, atualmente, será difícil encontrar sites que possuam apenas elementos HTML. Dessa forma, é possível considerar o HTML como o "esqueleto" da sua estruturação.

Suponha que além desse esqueleto, é preciso organizar o corpo. Para isso, existem as linguagens CSS e o JavaScript, que quando combinadas com o HTML, formam a base para os sites atuais.

De que modo o HTML funciona?

(Voltar)

Através de um documento HTML, que possui a extensão .html, o navegador efetua a leitura do arquivo e reproduz o seu conteúdo afim de que o usuário possa visualizá-lo. Os arquivos .html podem ser visualizados em qualquer navegador (Google Chrome, Safari, ou Mozilla Firefox).

É comum que os sites sejam compostos por diversas páginas HTML, por exemplo: suponha que um site possua três páginas, uma homepage, uma página de contato e uma página de produtos. Esse website terá, pelo menos, três documentos .html distintos, estando relacionados a cada uma das páginas do website.

É possível escrever o código a partir de qualquer editor de texto, como o bloco de notas. Cada página será composta de uma sére de tags, que podem ser consideradas como os blocos de construção dessas páginas. Dessa forma, essa estruturação é a maneira que o HTML marca os conteúdos, criando a hierarquia e a estrutura do mesmo, dividido entre seções, parágrafos, cabeçalhos, e etc.

Como utilizar as tags

(Voltar)

Por meio de editores de texto, como o Sublime Text, o NotePad++ ou até mesmo o bloco de notas, é possível estruturar um documento com a extensão .html que será reproduzido pelos navegadores.

Como foi explicado anteriormente, cada um dos documentos é composto por uma série de tags, que são códigos que definem toda a organização da página, bem como seu tamanho, a fonte da letra, as cores, as quebras de linha e etc. Grande parte dos elementos do documento HTML são compostos por uma estrutura de abertura e outra de fechamento.
